# GeekCoin: Coin-System

  A system for sorting and monitoring peso coins through a machine to a website. Demo: [Geek Coin](https://www.youtube.com/watch?v=LxUQS6OEqfE)
## Technology Used
### Machine
- Raspberry Pi 0 w (Python)
- Arduino (C++)
### Website
- HTML
- CSS/Bootstrap
- Javascript/jQuery
- PHP
